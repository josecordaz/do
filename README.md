## Install

```shell
make install
```

## Delete empty and not empty folder

```shell
run del FOLDER
```

## Get folder size

```shell
run size FOLDER
```

## Git add, commit, push
```shell
run git MESSAGE
```

## Delete all node_modules folders recursively
```shell
run nm FOLDER
```

## Import sql zip file to mysql server from a folder
```shell
go run main.go mysql --folder ~/Desktop --filePassword "PASSWORD"
```
